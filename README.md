
# Barthez

Barthez is a Rust implementation of the DNS guide from
[here](https://github.com/EmilHernvall/dnsguide/blob/master/chapter1.md).

# Things to learn

- Multithreading using async/await/Tokio
- How DNS works
- Dockerise a program
